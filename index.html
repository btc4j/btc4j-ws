Bitcoin Java Web Services

Open source SOAP interface to bitcoind API.

Development build status:

btc4j-ws uses:

  • Bitcoin Java Daemon Wrapper (org.btc4j.daemon)
  • Spring Framework (org.springframework)
  • Apache CXF (cxf.apache.org)
